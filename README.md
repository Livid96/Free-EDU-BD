🎓 FREE-EDU | Honours Portal
FREE-EDU is a lightweight, high-performance web portal designed to centralize academic resources for students. This project aims to break barriers in education by providing a free, open-source alternative to paid learning management systems.

✨ Features
Smart Navigation: Auto-hiding navbar on scroll for a focused reading experience.

Dynamic Search: Real-time filtering of subjects and faculties.

Modern UI: Glassmorphism design with a dark-mode-first "Ambient" aesthetic.

Responsive: Fully optimized for mobile, tablet, and desktop users.

Organized Structure: Clean, folder-based repository for easy subject management.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Modern Flexbox/Grid), Vanilla JavaScript.

Fonts: Inter, Syncopate, and Plus Jakarta Sans.

Hosting: GitHub Pages.

📂 How it's Organized
The repository follows a clean folder structure where each subject (e.g., Economics, Physics) resides in its own directory for better maintenance and URL readability.
